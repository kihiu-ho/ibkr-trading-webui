WHAT YOU NEED TO DO:
═══════════════════════════════════════════════════════════════

The webapp needs PostgreSQL and Redis to run.

CHOOSE ONE METHOD:

┌─────────────────────────────────────────────────────────────┐
│ METHOD 1: Docker (Recommended)                              │
└─────────────────────────────────────────────────────────────┘

  1. Start Docker Desktop (from Applications)
  
  2. Run: ./start-services.sh
  
  3. Run: ./start-webapp.sh
  
  4. Open: http://localhost:8000/workflows


┌─────────────────────────────────────────────────────────────┐
│ METHOD 2: Homebrew (No Docker needed)                       │
└─────────────────────────────────────────────────────────────┘

  1. Install:
     brew install postgresql@15 redis
  
  2. Start:
     brew services start postgresql@15
     brew services start redis
  
  3. Setup database:
     createdb ibkr_trading
     psql ibkr_trading -f database/init.sql
  
  4. Run: ./start-webapp.sh
  
  5. Open: http://localhost:8000/workflows


VERIFY EVERYTHING:
═══════════════════════════════════════════════════════════════

Run diagnostics:
  ./check-services.sh

Should show:
  ✓ PostgreSQL running on port 5432
  ✓ Redis running on port 6379
  ✓ .env file exists
  ✓ Virtual environment exists
