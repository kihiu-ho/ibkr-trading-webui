Connection Instructions

  1. Start the Gateway: If offline, run:
    docker compose up ibkr-gateway
  2. Initial Login: Click "Open Gateway Login" above and authenticate with your IBKR credentials
  3. Two-Factor Authentication: Approve the connection on your IBKR mobile app
  4. Verify Connection: Return to this page and click "Refresh" to verify authentication
  5. Alternative: Use the "Connect to Gateway" button below to trigger re-authentication via API
Troubleshooting:
  • Ensure ports 5055 and 5056 are not blocked by firewall
  • Check Docker logs: docker logs ibkr-gateway
  • Gateway may take 30-60 seconds to fully start
